Show All Categories in Verticall Drop&down menu.
== Description ==
 * Show All Categories With All Sub-Category.
 * You Can Change Direction
 * You Can Select 12 theme
 
 <a href="http://ctboard.com/">Live Demo</a>

== Installation ==

1. Upload plugin From wordpress admin and Active it.
2. Go to Setting >> Vertical menu and Set options
3. Go to widget and Drag&Drop Vertical menu widget in You theme Position.
